## 🚀 セットアップ手順

### 1. Googleフォームとスプレッドシートの準備

1. 上記の設問でGoogleフォームを作成
2. フォームの「回答」→「スプレッドシートにリンク」でスプレッドシートを作成

### 2. Google Apps Scriptの設定

1. スプレッドシートを開く
2. 「拡張機能」→「Apps Script」を開く
3. `Code.gs` の内容を貼り付けて保存
4. `CONFIG` の `GITHUB_TOKEN` / `GITHUB_OWNER` / `GITHUB_REPO` を確認（設定済み）

### 3. トリガーの設定

1. Apps Script エディタ左メニュー「トリガー」→「トリガーを追加」
2. 実行する関数: `onFormSubmit`
3. イベントのソース: スプレッドシートから
4. イベントの種類: フォーム送信時
5. 保存

### 4. 動作確認

1. フォームからテスト送信
2. `https://good-vibes-agency.github.io/gva-briefs/briefs/[スラグ]/` にアクセス
3. ページが表示されれば成功 🎉

## 📄 生成されるURL構造

```
https://good-vibes-agency.github.io/gva-briefs/         ← 一覧
https://good-vibes-agency.github.io/gva-briefs/briefs/phoebe-2026-06/   ← 個別ページ
```

## 💡 カスタマイズ

- `Code.gs` 内の `TEMPLATE` 変数でHTMLデザインを変更可能
- FAQは最大10件まで（FAQ質問1〜FAQ質問10）
- 過去の投稿例は改行区切りで複数設定可能
